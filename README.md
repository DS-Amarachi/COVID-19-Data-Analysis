# COVID-19 Data Analysis Project

This project analyzes COVID-19 data in relation to various happiness indicators across different countries. The analysis explores relationships between:
- COVID-19 infection rates
- COVID-19 death rates
- GDP per capita
- Social support
- Healthy life expectancy
- Freedom to make life choices

## Dataset
The analysis uses three main datasets:
1. COVID-19 confirmed cases dataset
2. COVID-19 deaths dataset
3. World Happiness Report dataset

## Key Findings
- Strong correlation between infection rates and death rates
- Higher rates in developed countries with better reporting systems
- Significant relationship with social support and life expectancy
- Impact of healthcare systems and population demographics

## Tools Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib

## How to Use
1. Clone the repository
2. Make sure you have all required Python packages installed
3. Run the Jupyter notebook to see the analysis
