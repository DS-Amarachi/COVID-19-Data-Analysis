# 🦠 COVID-19 Data Analysis Project

A comprehensive data analysis project examining the relationship between COVID-19 patterns and various happiness indicators, economic factors, and health metrics across countries.

## 📊 Overview

This project analyzes COVID-19 data in relation to various happiness indicators across different countries. The analysis explores relationships between:
- COVID-19 infection rates
- COVID-19 death rates
- GDP per capita
- Social support
- Healthy life expectancy
- Freedom to make life choices

## 📁 Datasets

The analysis uses three main datasets:
1. **COVID-19 Confirmed Cases** - `covid19_Confirmed_dataset.csv`
2. **COVID-19 Deaths** - `covid19_deaths_dataset.csv`
3. **World Happiness Report** - `worldwide_happiness_report.csv`

## 🔍 Key Findings

- 📈 Strong correlation between infection rates and death rates
- 🏥 Higher rates in developed countries with better reporting systems
- 🤝 Significant relationship with social support and life expectancy
- 💊 Impact of healthcare systems and population demographics

## 🛠️ Tools & Libraries

- **Python** 3.x
- **Pandas** - Data manipulation and analysis
- **NumPy** - Numerical computing
- **Seaborn** - Statistical data visualization
- **Matplotlib** - Data visualization

## 🚀 Getting Started

### Prerequisites
```bash
pip install pandas numpy seaborn matplotlib jupyter
```

### Usage
1. Clone the repository
   ```bash
   git clone https://github.com/DS-Amarachi/COVID-19-Data-Analysis.git
   cd COVID-19-Data-Analysis
   ```
2. Install required dependencies using the requirements.txt file
3. Open and run the Jupyter notebook
   ```bash
   jupyter notebook covid19_data_analysis.ipynb
   ```
